# AdNet Development Guide

This guide will help you set up and run each of the AdNet applications locally. Make sure you have Node.js (version 16.x or 18.x) installed on your system before proceeding.

## Seller AdNet Desktop

<Card title="GitHub Repository" icon="github" href="https://github.com/ondc-projects/Adnet-Desktop">
  View the source code on GitHub
</Card>

To set up the Seller AdNet Desktop application:

1. Clone the repository:
   ```bash
   git clone https://github.com/ondc-projects/Adnet-Desktop.git
   cd Adnet-Desktop
   ```

2. Install dependencies:
   ```bash
   yarn install
   ```
   or
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   yarn dev
   ```
   or
   ```bash
   npm run dev
   ```

The application should now be running on `http://localhost:3000`.
